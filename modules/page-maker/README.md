# Ant Extensions - Page Maker

### Page maker for building dashboards

---

### Install

```shell
# Using npm
npm install @ant-extensions/page-maker

# Using yarn
yarn add @ant-extensions/page-maker
```

---

### Basic Usage

```tsx
import { PageMaker } from "@ant-extensions/page-maker";

export const Tester = () => {
  return <div />;
};
```
